# Amazing Hand 中文快速验证使用说明

## 🤖 项目简介

Amazing Hand 是一个开源的8自由度仿人机械手项目，具有以下特点：
- **4根手指**，每根手指2个关节
- **8个自由度**（每根手指2个：弯曲/伸展 + 外展/内收）
- **所有驱动器内置**，无需外部缆线
- **3D打印**制作，成本低廉（<200欧元）
- **重量仅400g**

## 📋 准备工作清单

### 硬件要求
- [ ] **Amazing Hand 机械套件**（3D打印件 + 电子元件）
- [ ] **Feetech SCS0009 舵机** × 8个
- [ ] **外部电源**：5V/2A DC电源适配器
- [ ] **控制器**（以下二选一）：
  - Arduino + Feetech TTL Linker
  - Waveshare 串行总线驱动器 + 计算机

### 软件要求
- [ ] **Arduino IDE**（如果使用Arduino控制）
- [ ] **Python 3.x** + **rustypot库**（如果使用Python控制）
- [ ] **Feetech调试工具**：用于设置舵机ID

## ⚙️ 快速组装步骤

### 1. 舵机ID设置（重要！）

**右手舵机ID分配**：
```
手指1：ID=1,2  （拇指）
手指2：ID=3,4  （食指）
手指3：ID=5,6  （中指）
手指4：ID=7,8  （无名指）
```

**设置方法**：
1. 使用Feetech调试软件连接单个舵机
2. 按照上述ID分配逐一设置每个舵机
3. 记录哪个物理位置对应哪个ID

### 2. 硬件连接

#### Arduino方案：
```
Arduino RX ↔ TTL Linker TX
Arduino TX ↔ TTL Linker RX
VCC ↔ 5V外部电源
GND ↔ 共地
```

#### Python方案：
```
串口总线驱动器 ↔ 计算机
外部电源 ↔ 舵机供电
```

### 3. 基础校准

首次使用前需要校准每个手指的中位位置：
- 运行校准程序调整`MiddlePos`参数
- 确保手指能够自然伸直和弯曲

## 🚀 快速验证步骤

### 方法一：Arduino控制

1. **打开Arduino IDE**
2. **安装SCServo库**：
   - 库管理器中搜索"SCServo"
   - 点击安装
3. **上传示例程序**：
   ```cpp
   // 文件：ArduinoExample/Amazing_Hand_Demo.ino
   // 确保选择正确的串口号和波特率
   ```
4. **连接外部电源**
5. **观察机械手动作**

### 方法二：Python控制（推荐）

1. **安装rustypot库**：
   ```bash
   pip install rustypot
   ```

2. **修改串口号**：
   ```python
   # 编辑 PythonExample/AmazingHand_Demo.py
   c = Scs0009PyController(
           serial_port="COM11",  # 修改为你的��际串口号
           baudrate=1000000,
           timeout=0.5,
       )
   ```

3. **校准中位值**：
   ```python
   # 根据你的组装情况修改这些值
   MiddlePos = [3, 0, -5, -8, -2, 5, -12, 0]
   ```

4. **运行演示程序**：
   ```bash
   cd PythonExample
   python AmazingHand_Demo.py
   ```

## 🎯 演示动作说明

程序运行后，机械手将依次执行以下动作：

1. **OpenHand()** - 张开手掌
2. **CloseHand()** - 握紧拳头
3. **OpenHand_Progressive()** - 渐进式张开
4. **SpreadHand()** - 手指展开
5. **ClenchHand() - 紧握
6. **Index_Pointing()** - 食指指向

## 🔧 常见问题解决

### 舵机不响应
- 检查外部电源连接（确保5V/2A供电）
- 确认串口号正确
- 验证舵机ID设置是否正确

### 手指动作不正常
- 重新校准`MiddlePos`参数
- 检查机械组装是否有卡阻
- 确认舵机角度范围设置

### Python报错
```bash
# 串口被占用
# 解决方法：关闭其他串口监视器

# rustypot库未安装
# 解决方法：pip install rustypot
```
